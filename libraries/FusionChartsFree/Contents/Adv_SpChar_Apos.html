Using Apostrophe Sign on chart

 
When using dataURL method
When using dataURL method, you need to encode apostrophe as ' as under:
<graph decimalPrecision='0' >
   <set name='John&apos;s Count' value='420' color='AFD8F8' />
   <set name='Mary&apos;s Count' value='295' color='F6BD0F' />
   <set name='Tom&apos;s Count' value='523' color='8BBA00' />
</graph>

When using dataXML method
When using dataXML method, you need to encode apostrophe character to %26apos; - else, you'll get an "Invalid XML error". Following is the full HTML Code to embed the chart:

<div id="chart1div">
   This text is replaced by the chart.
</div>
<script type="text/javascript">
   var chart1 = new FusionCharts("FCF_Column2D.swf", "ChId1", "350", "200");
   chart1.setDataXML("<graph decimalPrecision='0' >
   <set name='John%26apos;s Count' value='420' color='AFD8F8' />
   <set name='Mary%26apos;s Count' value='295' color='F6BD0F' />
   <set name='Tom%26apos;s Count' value='523' color='8BBA00' />
   </graph>");
   chart1.render("chart1div");
</script>
You'll again get the same output as before.
